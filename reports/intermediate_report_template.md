# Промежуточный отчёт по магистерской работе

## Тема

«Разработка систем идентификации и трекинга подвижных объектов в системе видеонаблюдения с перекрывающимися полями обзора»

## 1. Цель текущего этапа

Целью текущего этапа являлась разработка воспроизводимого программного прототипа детекции и однокамерного трекинга людей, который в дальнейшем будет использован как основа модуля межкамерного объединения траекторий.

## 2. Выполненные работы

На текущем этапе:
- сформулирована модульная архитектура системы;
- выбран детектор семейства YOLO;
- подготовлены конфигурации алгоритмов ByteTrack и BoT-SORT;
- реализовано сохранение локальных траекторий в табличном формате;
- реализовано сохранение визуализированного видеопотока;
- реализован сбор параметров и результатов каждого эксперимента;
- подготовлен протокол сравнительного исследования трекеров.

## 6. Промежуточный вывод

По результатам первого этапа создан базовый прототип, обеспечивающий обнаружение людей, присвоение локальных идентификаторов и сохранение траекторий. Сравнение ByteTrack и BoT-SORT позволило определить исходный алгоритм для последующей разработки межкамерной ассоциации. Полученные результаты являются базовой линией, относительно которой далее будет оцениваться вклад визуальных и пространственно-временных признаков.

## 7. Дальнейшая работа

Следующий этап включает:
1. извлечение ReID-признаков для локальных траекторий;
2. формирование tracklet-представлений;
3. сопоставление траекторий между двумя камерами;
4. добавление временных и геометрических ограничений;
5. оценку IDF1, HOTA и числа ID switches на размеченных данных.
