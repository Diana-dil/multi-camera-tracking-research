# Данные

Большие видео и датасеты не хранятся в Git.

Для первого запуска положите короткое видео с людьми по адресу:

```text
data/samples/people.mp4
```

Рекомендуемый первый ролик:
- 20–60 секунд;
- статичная камера;
- 2–10 человек;
- хотя бы одно частичное перекрытие людей;
- без персональных или закрытых производственных данных.

Позднее здесь появятся адаптеры для MOT17 и WILDTRACK.
