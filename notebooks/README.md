# Notebooks

Позднее здесь будет ноутбук для:
- чтения `observations.csv` и `tracks_summary.csv`;
- построения распределений длительности треков;
- сравнения ByteTrack и BoT-SORT;
- визуализации ошибок ассоциации.

Первый этап намеренно реализован скриптами, чтобы эксперименты были воспроизводимыми, а не зависели от состояния ячеек ноутбука.
